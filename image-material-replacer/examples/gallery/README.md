# 示例画廊 | Example Gallery

本目录用于存放使用"图片素材替换生成器"生成的示例图片。

## 📁 目录结构

```
gallery/
├── ecommerce/          # 电商产品图示例
├── social-media/       # 社交媒体配图示例
├── advertising/        # 广告创意示例
└── brand-upgrade/      # 品牌升级示例
```

## 📸 添加你的作品

如果你使用本技能生成了优质图片，欢迎分享到这里！

### 提交格式

每个示例应包含：
1. **生成的图片**（PNG/JPG）
2. **示例图片**（参考图）
3. **素材图片**（上传的素材）
4. **提示词文本**（used-prompt.txt）
5. **说明文档**（README.md）

### 示例目录结构

```
gallery/ecommerce/example-01/
├── output.png          # 生成的最终图片
├── reference.png       # 示例参考图
├── material.png        # 使用的素材
├── prompt.txt          # 使用的完整提示词
└── README.md           # 说明文档
```
